## 📌 Overview

Surf Athlete is a cross-platform training application designed specifically for surfers. It delivers personalised workout plans, tracks athlete performance, and helps surfers of all levels improve their fitness and technique — available on Web, iOS, and Android.

---

## ✨ Key Features

- 🏋️ Personalised surf-specific workout plans
- 📊 Performance tracking and progress analytics
- 🌊 Training programs tailored for all skill levels
- 📱 Cross-platform — Web, iOS & Android
- 🎯 Goal setting and achievement tracking
